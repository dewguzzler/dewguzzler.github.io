Latest Projects

Twitch IRC client

An application to connect to the Twitch IRC

Android

Twitch.tv is the leading website, recently purchased by Amazon, that allows gamers to stream themselves playing games to a worldwide audience. Twitch has provided a way for streamers to interact with the viewers by hosting their own IRC server.

This application utilizes OAuth and Twitch's API to connect a user to any valid Twitch IRC channel. It let's users chat and see who else is in the chat room. It also mimics Twitch's chat on their website by replacing certain keywords with their emoticons.

Decimal.time Open Source

Decimal.time is a watch face for the popular Pebble smartwatch. Using a time convention that was used by the Resistance during the French Revolution, based on a 10 hour day, it displays the current time using C and JavaScript.
